## Démarrage rapide

Quatre étapes te mènent d'un projet vide à des fichiers de langue traduits. La CLI est la voie rapide ; tout ce qu'elle fait est aussi disponible depuis [le SDK](/docs/sdk).

### 1. Installe

verbatra est une dépendance de développement. Installe-la et lance le binaire via ton gestionnaire de paquets.

```bash
pnpm add -D @verbatra/cli
# npm install -D @verbatra/cli
# yarn add -D @verbatra/cli
```

Nécessite Node.js `>=22.14.0`. Invoque-la avec `pnpm verbatra ...`, `npx verbatra ...` ou `yarn verbatra ...`.

### 2. Génère une configuration

`init` écrit un `verbatra.config.ts`, un `.env.example`, et ajoute ton `.env` au gitignore pour qu'une vraie clé n'atterrisse jamais dans un commit.

```bash
verbatra init --provider gemini
```

Avec un terminal attaché, il te demande tout ce qu'il ne peut pas définir par défaut ; ajoute `--yes` pour prendre les valeurs par défaut (source `en`, cible `de`, fichiers `locales/{locale}.json`) et sauter les questions.

### 3. Définis ta clé API

Les clés proviennent de l'environnement, jamais de la configuration. Chaque fournisseur lit exactement une variable. Pour Gemini, c'est `GEMINI_API_KEY`.

```bash
export GEMINI_API_KEY=your-key-here
```

verbatra charge `.env.local` puis `.env` depuis le répertoire de travail avant de s'exécuter, et une variable déjà présente dans ton shell l'emporte toujours. Consulte [Fournisseurs](/docs/providers) pour la variable de chaque fournisseur.

<Callout title="Petit projet ou projet perso ? Utilise Gemini gratuitement.">
Gemini dispose d'un palier d'API réellement gratuit, ce qui en fait la façon la plus simple de commencer sans aucun coût. Récupère une clé sur [Google AI Studio](https://aistudio.google.com/apikey), définis `GEMINI_API_KEY` et lance `verbatra init --provider gemini`. Le palier gratuit a des limites de requêtes par minute et par jour, alors étale une grande première traduction (une langue à la fois suffit largement). Consulte [Fournisseurs](/docs/providers) pour les compromis face à Anthropic, OpenAI et DeepL.
</Callout>

### 4. Traduis

```bash
verbatra translate
```

verbatra lit la langue source, la compare à ce qu'il a traduit la dernière fois, et envoie uniquement les chaînes nouvelles et modifiées à Gemini. Chaque fichier cible est écrit sur place, à côté de ta source. Relance-le sans rien modifier et rien n'est envoyé : chaque clé est déjà à jour.
