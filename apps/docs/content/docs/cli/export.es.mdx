---
title: verbatra export
description: Exporta las cadenas sin traducir a un libro de Excel con estilos para un traductor humano.
---
Escribe las cadenas que necesitan traducción en un libro de Excel con estilos que un humano puede rellenar y devolver para [`verbatra import`](/docs/cli/import). Por defecto el libro lleva solo las cadenas faltantes y modificadas: las mismas claves que una ejecución automática enviaría a un proveedor. `export` nunca llama a un proveedor y no necesita clave API.

## Sinopsis

```bash
verbatra export [flags]
```

## Flags

| Flag | Argumento | Por defecto | Efecto |
| --- | --- | --- | --- |
| `--cwd` | `<path>` | directorio actual | resuelve la configuración y los archivos de locale desde este directorio |
| `--config` | `<path>` | busca uno | carga este archivo de configuración en lugar de buscar uno |
| `--out` | `<path>` | `verbatra-translations.xlsx` | escribe el libro en esta ruta |
| `--locales` | `<list>` | todos los destinos configurados | subconjunto de locales de destino separado por comas a exportar |
| `--include-unchanged` | ninguno | desactivado | exporta también las cadenas que ya están al día |
| `--json` | ninguno | desactivado | imprime el resultado de la exportación (ruta y número de filas por locale) como un objeto JSON en stdout; los errores siguen yendo a stderr |

Un valor `--locales` vacío o desconocido se rechaza con salida `2` en lugar de exportar nada en silencio.

## Ejemplos

```bash
# write the workbook with missing and changed strings
verbatra export

# only the German and French sheets, to a specific path
verbatra export --locales de,fr --out handoff/round-2.xlsx
```

## Códigos de salida

| Código | Significado |
| --- | --- |
| `0` | el libro se escribió |
| `2` | no se pudo ejecutar: un error de toda la ejecución, o un error de uso (incluido un valor `--locales` vacío o desconocido) |

`export` no tiene modo de fallo por locale, así que nunca sale con `1`.

## Relacionado

- [Traducción humana](/docs/manual-translation) cubre la disposición del libro, las columnas de revisión y el ciclo completo de ida y vuelta.
- [`verbatra import`](/docs/cli/import) lee de vuelta el libro rellenado.
