## Inicio rápido

Cuatro pasos te llevan de un proyecto vacío a archivos de locale traducidos. La CLI es el camino rápido; todo lo que hace también está disponible desde [el SDK](/docs/sdk).

### 1. Instalar

verbatra es una dependencia de desarrollo (consíguela con la tarjeta de instalación de arriba, o con `pnpm add -D @verbatra/cli`). Requiere Node.js `>=22.14.0`, y ejecutas el binario a través de tu gestor de paquetes: `pnpm verbatra ...`, `npx verbatra ...` o `yarn verbatra ...`.

### 2. Generar una configuración

`init` escribe un `verbatra.config.ts`, un `.env.example` e ignora en git tu `.env` para que una clave real nunca acabe en un commit.

```bash
verbatra init --provider gemini
```

Con una terminal conectada, pregunta por cualquier cosa que no pueda establecer por defecto; añade `--yes` para tomar los valores por defecto (origen `en`, destino `de`, archivos `locales/{locale}.json`) y saltarte las preguntas.

### 3. Configura tu clave API

Las claves provienen del entorno, nunca de la configuración. Cada proveedor lee exactamente una variable. Para Gemini es `GEMINI_API_KEY`.

```bash
export GEMINI_API_KEY=your-key-here
```

verbatra carga `.env.local` y luego `.env` desde el directorio de trabajo antes de ejecutarse, y una variable que ya esté en tu shell siempre gana. Consulta [Proveedores](/docs/providers) para ver la variable de cada proveedor.

<Callout title="¿Proyecto pequeño o proyecto personal? Usa Gemini gratis.">
Gemini tiene un nivel de API genuinamente gratuito, lo que lo convierte en la forma más fácil de empezar sin ningún coste. Consigue una clave en [Google AI Studio](https://aistudio.google.com/apikey), establece `GEMINI_API_KEY` y ejecuta `verbatra init --provider gemini`. El nivel gratuito tiene límites de solicitudes por minuto y por día, así que dosifica una primera traducción grande (un locale a la vez es más que suficiente). Consulta [Proveedores](/docs/providers) para ver las ventajas y desventajas frente a Anthropic, OpenAI y DeepL.
</Callout>

### 4. Traducir

```bash
verbatra translate
```

verbatra lee el locale de origen, lo compara con lo que tradujo la última vez y envía solo las cadenas nuevas y modificadas a Gemini. Cada archivo de destino se escribe en su sitio, junto a tu origen. Ejecútalo de nuevo sin editar nada y no se envía nada: cada clave ya está actualizada.
