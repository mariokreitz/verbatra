# Introduction

verbatra automates keeping your i18n translation files in sync. You maintain the source locale, and verbatra translates the others through the provider you choose, re-translating only the keys that changed since the last run.

These guides cover installing and configuring verbatra, the providers and formats it supports, how a run works under the hood, and using it from the command line, in CI, or through the SDK.
