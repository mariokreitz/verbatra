## Quick start

Four steps take you from an empty project to translated locale files. The CLI is the fast path; everything it does is also available from [the SDK](/docs/sdk).

### 1. Install

verbatra is a dev dependency (grab it with the install card above, or `pnpm add -D @verbatra/cli`). It needs Node.js `>=22.14.0`, and you run the binary through your package manager: `pnpm verbatra ...`, `npx verbatra ...`, or `yarn verbatra ...`.

### 2. Scaffold a config

`init` writes a `verbatra.config.ts`, a `.env.example`, and gitignores your `.env` so a real key never lands in a commit.

```bash
verbatra init --provider gemini
```

With a terminal attached it prompts for anything it cannot default; add `--yes` to take the defaults (source `en`, target `de`, files `locales/{locale}.json`) and skip the prompts.

### 3. Set your API key

Keys come from the environment, never the config. Each provider reads exactly one variable. For Gemini that is `GEMINI_API_KEY`.

```bash
export GEMINI_API_KEY=your-key-here
```

verbatra loads `.env.local` then `.env` from the working directory before it runs, and a variable already in your shell always wins. See [Providers](/docs/providers) for every provider's variable.

<Callout title="Small project or side project? Use Gemini for free.">
Gemini has a genuinely free API tier, which makes it the easiest way to start with zero cost. Grab a key from [Google AI Studio](https://aistudio.google.com/apikey), set `GEMINI_API_KEY`, and run `verbatra init --provider gemini`. The free tier has per-minute and per-day request limits, so pace a large first-time translation (one locale at a time is plenty). See [Providers](/docs/providers) for the trade-offs against Anthropic, OpenAI, and DeepL.
</Callout>

### 4. Translate

```bash
verbatra translate
```

verbatra reads the source locale, diffs it against what it translated last time, and sends only the new and changed strings to Gemini. Each target file is written in place, next to your source. Run it again without editing anything and nothing is sent: every key is already current.
